Bulk download

# Download everything (~31 GB)
python scripts/download-releases.py

# Or download specific format only
gh release download admin/states --pattern "*.parquet"
gh release download admin/states --pattern "*.pmtiles"

In-repo data

Smaller datasets (shapefiles, GeoJSON, KML) are stored directly in the data/ directory. Clone the repo or download individual files from GitHub.
